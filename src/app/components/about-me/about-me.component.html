About Me

If you are looking for someone who is going to sit in meetings or in an interview and talk technical terms then you've come to the wrong place. If you are looking for a hard working engineer with a thirst to learn more keep reading. I built this digital resume / portfolio to demonstrate some of the skills I know. The back end API is built with Spring Boot providing some basic rest endpoints and the front end is built using Angular 12. Both are hosted on Netlify. Angular Universal is used to load some of the resources server side to speed things up as well as the service worker that is used to control caching. The service worker is also used to turn this bad boy into a Progressive Web App. In case you are wondering about a specific skill you need and do not see please just shoot me a message at misterhedrick@gmail.com and I will be glad to take a look.
